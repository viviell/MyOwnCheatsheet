Visual Rules

Property descriptionDescription
Font-family The font-family property defines the typeface of an element.
Font-size The font-size property controls the size of text displayed.
Font-weight The font-weight property defines how thin or thick text is displayed.
Text-align The text-align property places text in the left. right or center od its parent container.
!important The !important flag will override any style.
